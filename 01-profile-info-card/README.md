
# Profile Info Card

## Description
This is a simple profile information card project.
In this project, I added my full name, role, location, and career objective.
The same information is also accessed using JavaScript and printed in the console.

## Technologies Used
- HTML
- CSS
- JavaScript

## What I Practiced
- How to access HTML elements using JavaScript
- How to read text content from HTML
- How to print data in the console

## Project Structure
- index.html – structure of the page
- style.css – styling of the page
- script.js – JavaScript file to access profile data
- README.md – information about the project

## Learning Outcome
This project helped me understand how JavaScript accesses data from HTML and how HTML, CSS, and JavaScript work together.
